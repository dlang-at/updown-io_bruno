# updown.io REST API — Bruno Collection

This is a collection containing **[updown.io](<https://updown.io/>)** REST API request templates
to be used with **[Bruno](<https://www.usebruno.com/>)**.

The official API documentation can be found here:
<https://updown.io/api>

## Getting started

Create a `.env` file supplying your *updown.io* API key as `API_KEY`.
See [.env.example](./.env.example) as a template.
